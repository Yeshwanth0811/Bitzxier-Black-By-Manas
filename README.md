### Prerequisites
- Node.js v16.9.0 or higher

```bash
git clone https://github.com/Manaseditz10/Bitzxier-Black
```

2. Install dependencies

- Create a `.env` file in the root directory
- Add your bot token and other configuration

```bash
npm start
```
